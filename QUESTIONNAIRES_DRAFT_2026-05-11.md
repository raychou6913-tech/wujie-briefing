# 7+1 份部門問卷草稿（題目清單 v2）

**版本：** 2026-05-11 v2（gg VAM 視角 review 後修正）
**目的：** 收集團隊腦袋裡的經驗常數，作為 ERP 系統預設值
**設計原則：** Wizard 一題一屏 / 最少打字 / 數字 + 選項為主 / 末頁開放補充
**目標填寫時間：** 每份 ≤ 20 分鐘

**🚨 標記說明：**
- `🚨 VAM 命脈題` = 答錯了 VAM 就算錯的關鍵題，**強制必填**
- ✓ = 一般必填
- 無標記 = 選填

## 設計筆記（v2 更新）

### gg review 後的主要改動
1. **金額分級** 取代範圍（Q1.10 / Q5.10-13）— 系統可直接取用具體數字
2. **新增成本軸題** — 採購不良率（1.16-17）/ 物流成本佔比（2.11-12）/ 電商退貨平台抽成（3.17-18）/ DTA 預扣稅（6.15）
3. **匯率 Text 改 Number**（6.15→6.16-19）— 系統可運算的具體數字
4. **送審輪數拆 2D / 3D**（原 3.9 → 3.13-14）
5. **實體樣品寄海外**（新 2.13）— 倒推算法的隱形時間
6. **新增問卷 8 IP 庫常數**（安然 + 珮玲）— MG / Royalty / 結算週期
7. **刪除無法計算的題**（原 Q7.7 歷史支出佔比）
8. **心情調查題併入開放補充**（Q4.8 / 5.12 / 6.17）— 減少填寫時間

### UI 規則（不變）
- 一題一屏，避免長 form 嚇人
- 進度條顯示 X / Y 題
- 答完一題自動進下一題（沒「下一題」按鈕）
- 中途離開 → localStorage 存進度
- 後端答案 POST 到飛書「系統常數設定」表

### 必填策略（v2 更新）
- 🚨 VAM 命脈題：強制必填，且輸入校驗（不能 0、不能空白、不能超出合理範圍）
