🌟 今日主推提案

Snoopy & 亞洲城市漫遊記 - 聯名盲盒

將 Snoopy 的經典形象與亞洲各城市的獨特風情及生活哲學相結合，透過新銳亞洲插畫師的筆觸，打造富有文化底蘊與收藏價值的全球旅人盲盒系列。

★★★★★

靈感來源：

近年來，潮玩市場對於具備文化敘事和藝術美感的產品需求日增。Snoopy 作為全球性 IP，其溫暖、幽默的形象非常適合融入不同文化背景。我們觀察到，Pop Mart 等品牌透過與藝術家合作，成功將盲盒從單純的玩具提升為藝術收藏品。此提案旨在借鑒這種模式，同時利用 Snoopy 在陸港澳台、東南亞、歐洲、紐澳、南美等廣泛授權市場的優勢，邀請台灣/韓國/新加坡等地擅長清新、療癒系風格的插畫師，描繪 Snoopy 在不同亞洲城市（如台北、首爾、曼谷、香港、新加坡）的特色場景，呈現多元的「亞洲漫遊」主題。

具體商品規格：

  • 尺寸： 8cm 高，搭配底座及可拆卸小配件。
  • 材質： PVC/ABS，細節塗裝，部分款式可加入植絨或透明件增加質感。
  • 售價區間： 批發價 USD 4.5 - 5.5 / 件。

目標市場與客群畫像：

  • 市場： 首發市場鎖定台灣、香港、新加坡，並逐步推廣至中國大陸、韓國及歐洲市場。
  • 客群： 20-35 歲，具有一定消費能力與藝術品味的白領女性、IP 收藏愛好者、旅行愛好者。她們注重生活品質，喜歡獨特、有故事性的商品，願意為設計感與限量感買單。

預估零售定價 + 批發毛利率：

  • 生產成本： 約 RMB 7 / 件 (USD 0.97)。
  • 建議批發價： USD 4.8 / 件 (約 RMB 34.56)。
  • 批發毛利率： (34.56 - 7) / 34.56 = 79.8%。
  • 預估零售定價： USD 12.99 - 14.99 / 件 (約 RMB 93.5 - 107.9)。此定價在高端盲盒市場具備競爭力。

VAM 貢獻潛力（具體數字估算）：

假設首年可批發銷售 150,000 件：

  • 總批發銷售額：150,000 件 * USD 4.8/件 = USD 720,000 (約 RMB 5,184,000)。
  • 總毛利貢獻：150,000 件 * (RMB 34.56 - 7)/件 = 150,000 件 * RMB 27.56/件 = RMB 4,134,000。

此提案有望為 Y1 的 VAM 淨利目標 RMB 608 萬帶來顯著的毛利貢獻，預計可覆蓋超過 60% 的年度淨利目標所需毛利。

上市時程建議：

  • 2026年Q2 (5-6月)： 藝術家鎖定與合作洽談、概念設計定稿。
  • 2026年Q3 (7-9月)： 3D 模型製作、打樣、生產。
  • 2026年Q4 (10月起)： 首批上市，配合節慶檔期及市場推廣。
目標市場: 台灣, 香港, 新加坡, 中國大陸, 韓國, 歐洲 商品類型: 聯名藝術家盲盒 VAM 貢獻潛力: 高 (Y1 > 60% 目標)

Mofusand x 治癒系寵物樂園 - 聯名盲盒

攜手中國大陸本土新銳插畫師，以 Mofusand 的萌趣形象為核心，結合生活中常見的療癒系寵物與場景，打造充滿溫馨與幽默感的盲盒系列，深化中國市場影響力。

★★★★★

靈感來源：

Mofusand 在中國大陸市場擁有極高人氣，其柔軟、慵懶的貓咪形象深受年輕族群喜愛。聯名藝術家盲盒在中國大陸市場有廣闊的潛力，例如 Pop Mart 旗下的 Molly、Labubu 等系列，都因其獨特的藝術風格和限定發售策略而大受追捧。此提案將選擇風格溫暖、擅長描繪動物與日常小確幸的中國大陸插畫師，透過其獨特的創作視角，為 Mofusand 注入更多生活化、情感化的故事，例如 Mofusand 與倉鼠、鸚鵡、小兔子等小動物的互動，或是其在咖啡廳、書房等場景中的慵懶姿態，吸引更廣泛的粉絲群體。

具體商品規格：

  • 尺寸： 5cm/8cm 混合尺寸，部分款式搭配小型場景配件。
  • 材質： PVC/ABS，高精度印刷與塗裝。
  • 售價區間： 批發價 USD 3.8 - 4.8 / 件。

目標市場與客群畫像：

  • 市場： 主力市場為中國大陸，可探討向東南亞市場擴展。
  • 客群： 18-30 歲，熱愛貓咪與萌系 IP 的學生、年輕白領、寵物愛好者。他們活躍於社群媒體，追求新潮、可愛的產品，喜歡收集系列商品。

預估零售定價 + 批發毛利率：

  • 生產成本： 約 RMB 6.5 / 件 (USD 0.90)。
  • 建議批發價： USD 4.0 / 件 (約 RMB 28.8)。
  • 批發毛利率： (28.8 - 6.5) / 28.8 = 77.4%。
  • 預估零售定價： USD 9.99 - 11.99 / 件 (約 RMB 71.9 - 86.3)。此定價策略旨在擴大銷量，提升市場佔有率。

VAM 貢獻潛力（具體數字估算）：

假設首年可批發銷售 250,000 件：

  • 總批發銷售額：250,000 件 * USD 4.0/件 = USD 1,000,000 (約 RMB 7,200,000)。
  • 總毛利貢獻：250,000 件 * (RMB 28.8 - 6.5)/件 = 250,000 件 * RMB 22.3/件 = RMB 5,575,000。

此提案由於市場體量大且 IP 本身熱度高，有望成為達成 VAM Y1 淨利目標 RMB 608 萬的關鍵驅動力，帶來超過 90% 的年度淨利目標所需毛利。

上市時程建議：

  • 2026年Q2 (5-6月)： 中國大陸插畫師篩選與簽約，設計概念深化。
  • 2026年Q3 (7-9月)： 產品開發、審批與生產。
  • 2026年Q4 (10月起)： 中國大陸市場預售及正式上市。
目標市場: 中國大陸, 東南亞 商品類型: 聯名藝術家盲盒 VAM 貢獻潛力: 極高 (Y1 > 90% 目標)
