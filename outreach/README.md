## 過濾機制（呼應 gg「濃縮+過濾」要求）

不要把 Apollo 拉到的 5000 個 lead 全發信 — 過濾規則：

1. **行業匹配**：玩具零售 / 文創通路 / 潮流商場 / 動漫經銷 / IP 採購商
2. **公司規模**：員工 50-500 人（太小沒採購力，太大流程慢）
3. **地區優先**：東南亞（雅加達展受眾）> 北美（已有合作）> 歐洲（德國展受眾）
4. **採購跡象**：LinkedIn 看到「buyer」「sourcing」「merchandising」title
5. **避雷**：已知合作 / 已拒絕 / 競爭對手

過濾後**月發信 100-300 封**（不是 5000 封），高品質高回覆率。
