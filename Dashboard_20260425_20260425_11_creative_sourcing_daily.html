本日創意焦點：數位周邊（AR / 數位收藏品）與實體公仔結合新方向

隨著數位科技的飛速發展和年輕一代消費習慣的轉變，單純的實體收藏品已不足以完全滿足市場需求。今日提案聚焦於如何將蕪解創意旗下強力 IP 的實體公仔，與 AR 互動體驗及數位收藏品概念結合，為產品線注入新的生命力，創造更高的附加價值與更豐富的用戶體驗，同時為公司 IPO 前的 VAM 目標貢獻潛力。

🌟 今日主推提案

1. 「Snoopy AR 世界旅人」系列盲盒公仔

實體公仔與沉浸式 AR 互動結合，將史努比的環球冒險帶入現實世界。

▼

靈感來源：

全球範圍內對 Snoopy/Peanuts IP 的廣泛喜愛，以及其「世界旅人」等多元主題的延展性。結合當前 AR 技術在消費領域的成熟應用，為收藏者提供超越實體的互動體驗。

具體商品規格：

  • 公仔尺寸/材質： 8cm PVC/ABS 材質精緻塗裝公仔，每款公仔設計為不同國家或城市的特色服飾及地標配件。
  • 數位整合：： 每個盲盒內含一個獨特的 QR Code 或 NFC 晶片，掃描後可解鎖專屬的 AR 應用程式或 WebAR 體驗。
  • AR 體驗內容：
    • AR 桌面寵物： 史努比虛擬形象可透過手機鏡頭出現在現實環境中，與用戶進行簡單互動（如跳舞、打招呼、與現實物品合影）。
    • 「城市探索」濾鏡：： 解鎖與公仔主題城市相關的 AR 濾鏡，豐富社交媒體分享內容。
    • 獨家數位收藏品： 首次拆盒可獲得一個該款公仔限定的「Snoopy 世界旅行護照」數位徽章（非區塊鏈 NFT，初期為應用內數位收藏品，預留升級區塊鏈 NFT 的可能性）。
  • 售價區間：
    • 預估生產成本： RMB 15/件 (含公仔本體 RMB 8 + AR 內容與晶片整合成本 RMB 7)。
    • 預估批發價： USD 6/件。
    • 預估零售定價： USD 18/件。

目標市場與客群畫像：

  • 市場： 歐洲、澳洲/紐西蘭、東南亞、香港、台灣、中國大陸。Snoopy 在這些地區擁有廣泛且忠實的粉絲基礎。
  • 客群： 18-35歲，喜愛潮流玩具、熱衷於探索新科技的年輕消費者與收藏者；重視社交媒體分享與獨特互動體驗的 IP 粉絲。

預估零售定價 + 批發毛利率：

  • 零售定價： USD 18/件 (約 RMB 129.6/件)。
  • 批發價： USD 6/件 (約 RMB 43.2/件)。
  • 批發毛利率： (RMB 43.2 - RMB 15) / RMB 43.2 = 65.28%。

VAM 貢獻潛力（具體數字估算）：

  • 預估首年全球銷量：200,000 件。
  • 預估首年批發銷售額：200,000 件 * USD 6/件 = USD 1,200,000 (約 RMB 8,640,000)。
  • 預估首年批發毛利貢獻：200,000 件 * (RMB 43.2 - RMB 15)/件 = RMB 5,640,000。
  • 此批發毛利貢獻將對 VAM 目標 (Y1 淨利 RMB 608萬) 有顯著支撐作用。

上市時程建議：

  • 概念驗證與技術合作夥伴鎖定： 2026年Q2
  • AR 內容開發與公仔設計定稿：： 2026年Q3
  • 生產與品控： 2026年Q4
  • 市場推廣與首發：： 2027年Q1 (春節/情人節檔期，利用其「旅行」主題)
目標市場：全球 (部分亞洲國家、歐洲、澳洲) 商品類型：AR 盲盒公仔 VAM 貢獻潛力：高

2. 「Chiikawa / Mofusand 數位潮玩勳章」系列盲盒

實體公仔搭配獨一無二的區塊鏈數位收藏品 (NFT)，開啟新世代潮玩收藏體驗。

▼

靈感來源：

Chiikawa 和 Mofusand 在中國大陸市場的超高人氣及社群活躍度。結合 NFT 區塊鏈技術在數位收藏品領域的興起，滿足年輕收藏者對獨特性、稀有性與數位資產所有權的追求。

具體商品規格：

  • 公仔尺寸/材質： 5cm PVC 材質 Q 版盲盒公仔，每款設計為 Chiikawa 或 Mofusand 的經典形象或限定造型。
  • 數位整合：： 每個盲盒內含一張獨立編號的卡片，卡片上印有可刮開的唯一密碼，用於兌換一個對應公仔的區塊鏈數位收藏品「潮玩勳章」(NFT)。
  • NFT 功能：
    • 每個 NFT 勳章擁有獨特的 ID 和稀有度標籤，可在專屬平台（或合作的第三方 NFT 市場）進行展示、交易。
    • 可預留未來作為線上活動入場券、解鎖獨家數位內容（如桌面背景、表情包）的憑證。
  • 售價區間：
    • 預估生產成本： RMB 8/件 (含公仔本體 RMB 6 + NFT 鑄造與卡片成本 RMB 2)。
    • 預估批發價： USD 4/件。
    • 預估零售定價： USD 12/件。

目標市場與客群畫像：

  • 市場： 中國大陸、香港、澳門。Chiikawa 和 Mofusand 在這些地區擁有龐大的粉絲基礎和高度的社群互動。
  • 客群： 15-30歲，熱衷於社交媒體、對新潮事物敏感、追求獨特性和收藏價值、對加密貨幣及 NFT 有一定認知或好奇的年輕潮玩愛好者。

預估零售定價 + 批發毛利率：

  • 零售定價：： USD 12/件 (約 RMB 86.4/件)。
  • 批發價：： USD 4/件 (約 RMB 28.8/件)。
  • 批發毛利率： (RMB 28.8 - RMB 8) / RMB 28.8 = 72.22%。

VAM 貢獻潛力（具體數字估算）：

  • 預估首年銷量：300,000 件。
  • 預估首年批發銷售額：300,000 件 * USD 4/件 = USD 1,200,000 (約 RMB 8,640,000)。
  • 預估首年批發毛利貢獻：300,000 件 * (RMB 28.8 - RMB 8)/件 = RMB 6,240,000。
  • 此批發毛利貢獻將顯著超越 Y1 淨利目標。

上市時程建議：

  • NFT 平台合作/開發與合規性研究： 2026年Q2
  • 公仔設計與 NFT 圖稿確認：： 2026年Q3
  • 生產與品控：： 2026年Q4初期
  • 市場預熱與首發：： 2026年Q4中旬 (抓住年底購物季與聖誕節檔期)
目標市場：中國大陸、香港、澳門 商品類型：NFT 盲盒公仔 VAM 貢獻潛力：極高

💡 延伸創意點

  1. 「IP 虛擬形象衣櫥」： 授權 IP（如 Opanchu, Muzik Tiger）實體掛件或小公仔，附贈兌換碼，可為用戶在合作的元宇宙平台或虛擬社交 App 中解鎖該 IP 的獨家虛擬服飾或配件，豐富用戶的虛擬形象。
  2. 「互動式數位故事書」： 購買 Maniani World 或飯糰與梅子主題的文具周邊（如筆記本、貼紙包），內含 QR Code 可連結至獨家互動式數位漫畫或短篇動畫，定期更新，提升內容粘性。
  3. 「盲盒開箱 AR 驚喜」： 為現有盲盒產品線升級，在 App 中增加「AR 開箱」功能，用戶掃描公仔後，可觸發一段有趣的 AR 動畫，模擬公仔「跳出來」的驚喜感，並解鎖該公仔的專屬數位卡片。

🏪 競品參考

  • 泡泡瑪特 (Pop Mart) / Labubu： 作為盲盒潮玩市場的領軍者，泡泡瑪特在實體產品設計和營銷上極具優勢，其限量、稀有款策略引發社群狂熱。目前已開始探索數位收藏品領域，推出「潮玩宇宙」App，內含數位卡牌及部分 NFT 項目，但尚未與實體公仔深度綁定，有待進一步觀察其數位化戰略。
  • Chiikawa (吉伊卡哇) 周邊： 在日本及亞洲市場擁有極高人氣，其周邊商品（包括盲盒、毛絨公仔、文具等）銷售火爆，社群互動熱烈。雖然其官方主要仍以實體商品為主，但粉絲自發的二次創作和數位分享已形成巨大聲量，顯示其 IP 具備強大的數位化潛力。
  • Mofusand 周邊：： 以其可愛的貓咪形象迅速走紅，在中國大陸及港台地區市場表現出色，周邊產品銷量可觀。與 Chiikawa 類似，Mofusand 的高社群參與度也為其數位內容和收藏品的發展奠定基礎。
  • Fuggler： 以其獨特搞怪的形象在全球範圍內吸引了一批另類收藏者。這類產品雖然風格迥異，但都證明了市場對獨特、有故事的收藏品有需求。在數位領域，Fuggler 或可探索 AR 互動的「惡搞」玩法。
