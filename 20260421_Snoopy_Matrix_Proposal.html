PART A｜情報席：路人魚解碼報告

A1 路人魚（Labubu）成功的真正原因

表面現象（大家都知道的）

  • Lisa（BLACKPINK）2024年公開背 Labubu 包掛，帶動全球瘋搶
  • Pop Mart 市值一年內翻倍，2024年股價最高漲幅超 300%
  • 二手市場隱藏款溢價 5-10 倍，形成「理財公仔」話題

深層成功邏輯（情報席分析）

  1. 醜萌美學崛起：消費者不要「完美」了
    • 2010年代：Hello Kitty / 迪士尼公主 = 完美無瑕可愛
    • 2020年代：Labubu / Mofusand / Chiikawa = 有缺陷、有個性、有「怪」感
    • 消費者洞察：完美讓人羨慕，不完美讓人共鳴。齜牙的 Labubu 比無缺點的公主更「真實」
    • 心理學根據：「不對稱美感」降低心理距離，觸發保護欲與親密感
  2. 情緒商品 > 功能商品
    • 購買動機從「它好看」→「它讓我感覺____」
    • Labubu 的核心情緒：「我也是一隻怪怪的小獸，但我很可愛」
    • Z世代的精神圖騰：反完美主義、自我接納、幽默看待焦慮
    • 花生漫畫有天然優勢：Charlie Brown 就是「失敗者英雄」的原型，比 Labubu 更有哲學深度
  3. 可見性設計：包包掛件是社交貨幣
    • Labubu 最爆的 SKU 不是桌上擺件，而是「包包掛件版」
    • 出門即展示，他人即傳播 → 行走的廣告牌
    • 購買行為研究：「我在 XX 地鐵看到一個人包上掛著 Labubu」→ 衝去買
    • 關鍵設計要求：正面造型辨識度高、尺寸適合掛件（6-8cm）、有掛鉤結構
  4. 盲盒機制 × 隱藏款 = 多巴胺設計
    • 不確定性觸發的多巴胺是確定性的 2.3 倍（神經科學研究）
    • 隱藏款比例 1/72 或 1/144 → 創造「傳說稀缺感」
    • 二手市場溢價 → 盲盒從「消費品」升格為「類投資品」
    • 開箱影片 UGC：一個人買，百萬人看開箱 → 免費流量飛輪
  5. KOL × IP 雙引擎
    • 路人魚 = Kasing Lung（藝術家品牌）× Pop Mart（渠道）× Lisa（名人引爆）
    • 三者缺一不可：IP 授權 + 渠道執行 + 名人點火
    • 花生漫畫對應：花生 IP（授權）× 蕪解（渠道執行）× 待定名人合作（東南亞/台灣 KOL 策略）

D3 業務席市場調研任務書

幕僚長交辦業務席，以下三個核心問題必須帶著腳去市場搜集真實反饋，不能只坐辦公室憑感覺作答。數據回來後幕僚長統整，才是定案的依據。

任務一：消費者現在喜歡什麼口味？

我們對消費者的理解停留在「喜歡 Snoopy」這個層次，不夠用。

▼ 展開詳情

業務席要向目標渠道（大陸精品玩具店、台灣誠品/蔦屋、泰國 Central 商圈）搜集：

  • 目前店內最暢銷的盲盒 IP 是哪三個？為什麼消費者買？
  • 消費者在店頭的口頭評語：是說「好可愛」還是「好有個性」？
  • 購買者年齡層與性別比實際觀察數字

回報格式：各市場各 3 家店，填回調研表，下週五前交幕僚長。

任務二：同時推幾個系列才不會把渠道撐死？

財務試算是一回事，渠道實際能吸收的鋪貨量是另一回事。

▼ 展開詳情

業務席要向現有合作客戶（大陸前 20 大客戶、台灣主要百貨買手）確認：

  • 一個新 IP 系列首批鋪貨量他們能接受多少件/店？
  • 若同時上兩個系列，他們的預算會怎麼分配？
  • 對花生漫畫盲盒系列的興趣程度（1-10 分，目前是未看過提案的冷評估）

回報格式：至少訪談 10 家客戶，回填意願評分表。

任務三：各市場的真實銷售規模上限是多少？

我們的財務模型是由上往下推的（市場規模 × 滲透率），業務席要做由下往上的驗證。

▼ 展開詳情

業務席要驗證：

  • 現有渠道客戶，對 Snoopy 盲盒系列願意下多少首批訂單？
  • 歐洲、紐澳市場：現有代理商關係是否能快速啟動？若無，開發新代理商的時間成本？
  • 機場免稅渠道：台灣桃園機場 + 曼谷蘇萬納蓬機場，進入條件與前置時間？

回報格式：渠道意願彙整表 + 潛在首批訂單量估算，30 天內完成。
