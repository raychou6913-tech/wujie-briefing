PART A｜情報席：路人魚解碼報告

A1 路人魚（海綿寶寶·比奇堡居民）成功的真正原因

正本清源：「路人魚」是《海綿寶寶》動畫中比奇堡的背景居民角色——那些在每一集反覆出現、有著誇張表情的無名魚類市民。他們不是主角，沒有名字，卻在 2020 年代成為盲盒市場的現象級爆款。這個案例對花生漫畫的啟示，遠比任何新 IP 都更直接。

路人魚為什麼會爆？

  • **① 「我認識這張臉！」的認知共鳴**
    • 全球有超過 20 億人看過海綿寶寶。比奇堡路人魚的誇張表情早已刻入大腦
    • 但在此之前，從來沒有人把他們做成「主角商品」
    • 消費者購買動機：「這個我認識！他是那個在背景驚叫的那隻魚！」
    • 這不是新 IP，是喚醒記憶——購買的是「與記憶重逢」的情感，不是「認識新角色」的好奇
  • **② 迷因（Meme）變商品：Z世代的購買語言**
    • 比奇堡路人魚的誇張表情早已是網路迷因素材庫
    • 「This is Fine」魚、「驚嚇路人魚」、「路人魚大叫」——每個梗都有數億次傳播
    • 當這些迷因臉被做成實體公仔，購買行為本身就是一個「迷因行為」
    • 社交媒體擴散邏輯：買到稀有款 → 曬圖說「我買到了那個 MEME！」→ 算法推流 → 更多人衝去買
  • **③ NPC 策略的顛覆性：配角成為主角**
    • 傳統 IP 商品邏輯：主角 > 配角 > 背景角色（背景角色幾乎不做）
    • 路人魚打破這個邏輯：**背景角色才是最稀缺的商品**
    • 稀缺性來自「大家都認識但誰也沒有」——比主角更難買到，更值得炫耀
    • 消費者說的話：「我已經有海綿寶寶了，但這隻路人魚我從沒見過做成公仔！」
  • **④ 完整世界觀的集齊慾**
    • 單一主角系列（只有海綿寶寶）無法觸發集齊慾
    • 整個比奇堡生態系（幾十隻路人魚）= 永遠集不完的收藏坑
    • 心理學：「接近完整」比「完全沒有」更讓人花錢——差一隻就是差一隻
    • 業務端效益：渠道補單率極高，消費者自己追著問「還有哪幾款沒出？」
  • **⑤ 低門檻進入 × 高門檻完整**
    • 第一隻：衝動購買（49-79 RMB，不貴）
    • 全系列：長期投入（幾十款 = 幾千 RMB）
    • 商業設計精髓：用低客單價引進門，用完整世界觀留住人

A4 競品快照（市場定位分析）

競品A：SpongeBob × 比奇堡路人魚系列

迷因認知 × NPC 稀缺性 × 完整世界觀，開創盲盒爆款新思路。

定價：RMB 59-89 盲袋

主打渠道：名創優品、潮玩集合店、電商

核心武器：

  • **迷因轉商品機制**：路人魚的誇張表情本身就是網路迷因，消費者購買是認同迷因文化。
  • **NPC 策略5點分析**：
    1. 低辨識度高熟悉度：大家認識但從未商品化。
    2. 去中心化：主角光環弱化，配角也能當主角。
    3. 反向稀缺：主角量大，配角反而少見。
    4. 社群擴散：買到特別的NPC更容易曬圖。
    5. 低成本高變現：形象簡單易生產，因迷因產生高溢價。
  • **比奇堡完整生態**：創造無限延伸的收藏坑，消費者永遠集不完。

對花生漫畫借鑑：花生宇宙深度優於比奇堡，但我們需要更主動地挖掘「背景角色」的故事與迷因潛力。

定價：JPY 660-2200（約 RMB 33-110）

主打渠道：便利店聯名、旗艦店、扭蛋

核心武器：

  • **療癒策略**：以「努力生活但又很弱小」的形象，引發廣大社畜共鳴，提供情感慰藉。
  • **高頻更新**：動畫、漫畫持續輸出新內容，保持 IP 熱度與新鮮感。

弱點與蕪解切入：

  • **語言壁壘**：內容依賴日文語境，東南亞市場滲透有限。
  • **蕪解切入**：花生漫畫角色（如 Charlie Brown）也有「失敗英雄」的療癒感，Snoopy 更有「白日夢逃避現實」的普世共鳴，無需語言即可傳達情緒，適合全球市場。

D3 業務席市場調研任務書

幕僚長交辦業務席，以下三個核心問題**必須帶著腳去市場搜集真實反饋**，不能只坐辦公室憑感覺作答。數據回來後幕僚長統整，才是定案的依據。

任務一：消費者現在喜歡什麼口味？

深入市場，挖掘消費者對盲盒 IP 的真實偏好。

我們對消費者的理解停留在「喜歡 Snoopy」這個層次，不夠用。
業務席要向目標渠道（大陸精品玩具店、台灣誠品/蔦屋、泰國 Central 商圈）搜集：

  • 目前店內最暢銷的盲盒 IP 是哪三個？為什麼消費者買？
  • 消費者在店頭的口頭評語：是說「好可愛」還是「好有個性」？
  • 購買者年齡層與性別比實際觀察數字

回報格式：各市場各 3 家店，填回調研表，下週五前交幕僚長。

任務二：同時推幾個系列才不會把渠道撐死？

平衡財務預算與渠道承載力，優化產品上市策略。

財務試算是一回事，渠道實際能吸收的鋪貨量是另一回事。
業務席要向現有合作客戶（大陸前 20 大客戶、台灣主要百貨買手）確認：

  • 一個新 IP 系列首批鋪貨量他們能接受多少件/店？
  • 若同時上兩個系列，他們的預算會怎麼分配？
  • 對花生漫畫盲盒系列的興趣程度（1-10 分，目前是未看過提案的冷評估）

回報格式：至少訪談 10 家客戶，回填意願評分表。

任務三：各市場的真實銷售規模上限是多少？

精準預估市場潛力，為長期擴張提供數據支撐。

我們的財務模型是由上往下推的（市場規模 × 滲透率），業務席要做由下往上的驗證：

  • 現有渠道客戶，對 Snoopy 盲盒系列願意下多少首批訂單？
  • 歐洲、紐澳市場：現有代理商關係是否能快速啟動？若無，開發新代理商的時間成本？
  • 機場免稅渠道：台灣桃園機場 + 曼谷蘇萬納蓬機場，進入條件與前置時間？

回報格式：渠道意願彙整表 + 潛在首批訂單量估算，30 天內完成。

E5 損益敏感性分析

風險情境模擬

評估極端情況下的財務表現，制定應對策略。

最大風險情境：若 8cm 系列銷售率僅達 70%（剩 30% 庫存）。

  • 世界旅人系列毛利影響：324,000件 × 0.7 (銷售率) × 18.69 (單件毛利) = RMB 423.83萬
  • 小黃鳥特攻隊毛利：RMB 252.31萬 (不受影響)
  • **總毛利** (風險情境)：RMB 423.83 + 252.31 = RMB 676.14萬
  • **固定成本合計**：RMB 250.2萬
  • **Y1 淨利 (風險情境)**：RMB 676.14 - 250.2 = RMB 425.94 萬

此處計算出的 RMB 425.94 萬 與提案中 RMB 277 萬 有差異，原因可能為固定成本或庫存處理成本計算方式不同。

對策：小黃鳥 5cm 掛件毛利更高（68%），作為壓艙石；加大電商尾貨消化管道。
