PART A｜情報席：路人魚解碼報告

A1 路人魚（Labubu）成功的真正原因

表面現象（大家都知道的）

  • Lisa（BLACKPINK）2024年公開背 Labubu 包掛，帶動全球瘋搶
  • Pop Mart 市值一年內翻倍，2024年股價最高漲幅超 300%
  • 二手市場隱藏款溢價 5-10 倍，形成「理財公仔」話題

深層成功邏輯（情報席分析）

① 醜萌美學崛起：消費者不要「完美」了

  • 2010年代：Hello Kitty / 迪士尼公主 = 完美無瑕可愛
  • 2020年代：Labubu / Mofusand / Chiikawa = 有缺陷、有個性、有「怪」感
  • 消費者洞察：完美讓人羨慕，不完美讓人共鳴。齜牙的 Labubu 比無缺點的公主更「真實」
  • 心理學根據：「不對稱美感」降低心理距離，觸發保護欲與親密感

② 情緒商品 > 功能商品

  • 購買動機從「它好看」→「它讓我感覺____」
  • Labubu 的核心情緒：「我也是一隻怪怪的小獸，但我很可愛」
  • Z世代的精神圖騰：反完美主義、自我接納、幽默看待焦慮
  • 花生漫畫有天然優勢：Charlie Brown 就是「失敗者英雄」的原型，比 Labubu 更有哲學深度

③ 可見性設計：包包掛件是社交貨幣

  • Labubu 最爆的 SKU 不是桌上擺件，而是「包包掛件版」
  • 出門即展示，他人即傳播 → 行走的廣告牌
  • 購買行為研究：「我在 XX 地鐵看到一個人包上掛著 Labubu」→ 衝去買
  • 關鍵設計要求：正面造型辨識度高、尺寸適合掛件（6-8cm）、有掛鉤結構

④ 盲盒機制 × 隱藏款 = 多巴胺設計

  • 不確定性觸發的多巴胺是確定性的 2.3 倍（神經科學研究）
  • 隱藏款比例 1/72 或 1/144 → 創造「傳說稀缺感」
  • 二手市場溢價 → 盲盒從「消費品」升格為「類投資品」
  • 開箱影片 UGC：一個人買，百萬人看開箱 → 免費流量飛輪

⑤ KOL × IP 雙引擎

  • 路人魚 = Kasing Lung（藝術家品牌）× Pop Mart（渠道）× Lisa（名人引爆）
  • 三者缺一不可：IP 授權 + 渠道執行 + 名人點火
  • 花生漫畫對應：花生 IP（授權）× 蕪解（渠道執行）× 待定名人合作（東南亞/台灣 KOL 策略）

D3 幕僚長最終建議

老董問的三個問題，幕僚長直接回答：

問：消費者現在喜歡什麼？
答：喜歡「有缺陷的可愛」、「能共鳴的失敗感」、「可展示的社交貨幣」。Snoopy 宇宙天然符合三條，但我們過去賣法太保守，沒有把這個故事說出來。

問：我們能推多少個系列？
