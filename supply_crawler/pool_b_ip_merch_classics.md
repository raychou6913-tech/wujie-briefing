# 池 B：IP 周邊主力品類（驗證好賣）

> 每週看新款。設計團隊主看，木子配合採購。
> 不是找新品類（那是池 C 的事），是找**現有品類的新款 / 新設計 / 新材質**。

---

## 子分類 + 關鍵字

### B1. 亞克力類（最賣）
**關鍵字**：亞克力吊飾 / 亞克力立牌 / 磁吸貼 / 桌擺 / 鑰匙圈 / 冰箱貼 / 透明 standee / acrylic charm / acrylic stand

**為什麼重要**：MOQ 低、生產快、IP 變現主力

### B2. 絨毛類
**關鍵字**：絨毛玩偶 / 包掛 / 抱枕 / 毛絨吊飾 / plush keychain / plush mascot

**為什麼重要**：女性買家最愛、客單價高

### B3. 印刷類
**關鍵字**：明信片 / 貼紙 / 海報 / 文件夾 / 萬年曆 / 卡套 / postcard / sticker pack

**為什麼重要**：MOQ 最低、毛利率不錯、易進文創通路

### B4. 3C 周邊
**關鍵字**：手機殼 / 卡套 / AirPods 套 / 充電線 / 手機支架 / phone case / AirTag holder

**為什麼重要**：年輕族群必備、新機型上市追時效

### B5. 文具
**關鍵字**：筆 / 筆袋 / 筆記本 / 便利貼 / 書籤 / stationery / notebook / pen pouch

**為什麼重要**：學生族群、低 MOQ、可塞展位

### B6. 生活雜貨
**關鍵字**：杯子 / 餐具 / 餐墊 / 抹布 / 香氛 / mug / coaster / scented candle

**為什麼重要**：客單價高、Lifestyle 訴求

### B7. 服飾配件
**關鍵字**：T 恤 / 帽 / 襪 / 包包 / 髮飾 / tee / cap / tote bag / hair accessory

**為什麼重要**：自我表達、客單可衝高（聯名款）

## 看新款的角度

不是看「新品類」，是看：

| 維度 | 範例 |
|------|------|
| **新外型** | 亞克力吊飾的「不規則異形」造型崛起 |
| **新材質** | 文件夾用毛氈替代塑膠 |
| **新印刷工法** | 立體浮雕 / 燙金 / 雷射切割 |
| **新尺寸** | mini 系列 vs 巨大化 trend |
| **新組合** | 手機殼 + 磁吸卡套 二合一 |

## 排除

- ❌ 已大量出 IP 的常規款（不算「新」）
- ❌ 普通 PVC 公仔（公司本來就在做）
- ❌ 太貴的客製化品（譬如金屬胸針 100 RMB 以上）

## 目標數量

- 月爬源：6 個平台（淘寶、Etsy、Amazon、Rakuten、Pinterest、小紅書）
- 月過濾後：50-80 個候選
- 週推設計團隊：Top 15（按品類分）
- 月推 Top 30 給「下次採購選品會議」

## 設計團隊對齊（5/15 會議）

5/15 跟設計團隊（Bevis / 詩彤 / 珮綾）開 30 分鐘：

1. 7 個子分類的優先順序？
2. 目前最缺哪個品類的新款靈感？
3. 看新款時想看什麼維度（外型 / 材質 / 工法 / 組合）？
4. 他們平常在哪裡找靈感（Behance / Pinterest / 巴塞爾 / 小紅書）？
5. 推送頻率 — 週 Top 15 太多還是剛好？
