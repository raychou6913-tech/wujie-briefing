# 池 D：新工法 / 新材質

> 設計團隊 + Bevis 主看。目標：給設計師「工具箱」— 看到新工法立刻聯想 IP 怎麼套用。
> 是「軟體 × 硬體」聖杯公式的「硬體」端。

---

## 關鍵字分類

### D1. 表面處理工法
| 關鍵字 | 應用 |
|--------|------|
| 燙金 / hot stamping / gold foil | 高貴感、節日感 |
| 燙印 / embossing | 立體浮雕觸感 |
| 立體浮雕 / 3D emboss | 同上 |
| 雷射切割 / laser cut | 精細鏤空 |
| 雷射雕刻 / laser engrave | 木質金屬深度刻 |
| 蝕刻 / etching | 金屬細節 |
| 噴砂 / sand blasting | 霧面質感 |
| 鏡面 / mirror finish | 反光高貴 |

### D2. 透明/半透明材質
| 關鍵字 | 應用 |
|--------|------|
| 亞克力透明 / clear acrylic | 已主流但仍在創新 |
| 樹脂滴膠 / resin coating | 透明保護 + 高貴感 |
| 環氧樹脂 / epoxy resin | 自製感、IP 客製 |
| 矽膠軟質 / soft silicone | 可愛感、抗摔 |
| 果凍料 / jelly material | 治癒感、年輕族群 |

### D3. 新材質
| 關鍵字 | 應用 |
|--------|------|
| 鋅合金 / zinc alloy | 高貴感 + 質感重 |
| 黃銅 / brass | Vintage + 文青 |
| 不鏽鋼 / stainless steel | 可持續訴求 |
| 軟質 PVC / soft PVC | 抗摔可彎 |
| TPR | 公仔常用，新色 / 新觸感 |
| 木質 + 環氧樹脂 | 高貴自然 |
| 毛氈 / felt | 暖系觸感 |

### D4. 印刷/染色新工法
| 關鍵字 | 應用 |
|--------|------|
| UV 印刷 | 立體 + 色彩飽和 |
| 數位印花 | 小量客製 |
| 雷射光澤 / holographic print | 變色彩虹（Y2K 風潮）|
| 螢光印刷 / glow-in-dark | 趨勢 + 互動 |
| 溫變色 / thermochromic | 互動感、孩童市場 |
| 變色龍油墨 / chameleon ink | 高貴 + 變色 |

### D5. 互動 / 機械工法
| 關鍵字 | 應用 |
|--------|------|
| 翻翻樂 / lenticular | 翻面變圖 |
| 旋轉機構 / rotating mechanism | 互動 |
| 磁吸組合 / magnetic snap | 模組化 |
| 拼接卡榫 / snap-fit | DIY 感 |
| 內含機關 / hidden mechanism | 驚喜開箱 |

## 過濾規則

| 過濾層 | 規則 |
|--------|------|
| **可量產** | 工廠有設備 + 報價合理 |
| **適合 IP** | 能讓 IP 角色加分（非純工業用途）|
| **成本可控** | 加工成本 < 單品售價 30% |
| **MOQ 合理** | 500-1000 件起即可（不要 10000 件起的）|

## 跟池 C 的搭配

池 C 找新主題 × 池 D 找新工法 = 「**軟體 × 硬體**」聖杯公式

例：池 C 找到「**美式公路旅行**主題上升」+ 池 D 找到「**鋅合金 + UV 印刷**新工法」→ Forge bedo 生提案「**美式公路旅行 × 史努比 × 鋅合金徽章**」→ 設計團隊不是憑空想，是拿兩塊拼圖拼裝。

這個搭配是公司護城河的核心。
