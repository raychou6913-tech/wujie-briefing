Chiikawa（吉伊卡哇）在中國大陸市場的病毒式增長與潛在風險並存

▼

Chiikawa 作為「電子布洛芬」在中國大陸年輕族群中持續爆紅，其療癒系形象與共鳴感極強的小人物故事，透過抖音與小紅書上的開箱、二創內容形成病毒式傳播。近期數據顯示，#吉伊卡哇盲盒 開箱影片本週新增超過 10 萬條，總播放量突破 5 億次，其中隱藏款公仔在二手市場溢價高達 200%。 深圳地區的 Chiikawa 快閃店單日人流峰值達 5,000 人次，平均客單價達 250 元人民幣，熱門產品持續補貨且實施限購措施。 然而，近期有報導指出，由於作者過往言論，部分中國大陸網友發起抵制聲浪，此雖未明顯影響銷售熱度，但需高度警惕其品牌形象的脆弱性。

對蕪解的影響：

Chiikawa 是蕪解創意在中國大陸市場的關鍵 IP，其極高熱度直接驅動盲盒公仔及掛件的銷售增長，遠超 Y1 淨利潤目標中對單一 IP 的預期貢獻。但潛在的輿論風險可能導致品牌形象受損，進而影響銷售及與授權方的關係。

建議行動及截止日：

立即啟動 Chiikawa 相關社群輿情 24 小時監測機制，重點關注抖音、小紅書、微博等平台上的負面聲量與抵制趨勢。4月23日前，提交首份輿情分析報告與應對預案。 加速 Chiikawa 具中國文化元素的限定商品開發（例如已在深圳快閃店推出的舞龍舞獅限定版），並在 5月15日前推出至少 1 款預售，以本土化策略強化 IP 在地連結，降低政治敏感度。

「真實感」與「用戶生成內容（UGC）」成為社群 IP 傳播的核心驅動力

▼

2026 年社群媒體內容策略研究指出，消費者對「真實感」的追求超越了純粹的「爆款」內容，並更傾向於與品牌建立「深度連結」，而非僅是「按讚」等輕量級互動。 短影音仍是主戰場，但「多圖+Reels」的組合拳更有效，尤其用戶生成的開箱影片、體驗分享、二創故事等內容，其互動率顯著高於品牌官方內容，成為 IP 病毒式傳播的關鍵。 抖音上「爆火卡通IP」的出現，往往伴隨著大量用戶自發性分享其收藏、改裝與情境演繹。

對蕪解的影響：

蕪解創意的 IP 周邊商品（盲盒、掛件）天然具備高度 UGC 潛力。若能有效引導並激勵用戶創作優質內容，將能以低成本實現高效的病毒式傳播，擴大 IP 影響力，直接促進批發與零售銷售。

建議行動及截止日：

針對 Mofusand 與 Opanchu 兩個 IP，在中國大陸（小紅書、抖音）與東南亞（TikTok、Instagram）市場，策劃為期一個月的「創意開箱挑戰」或「IP情境故事」UGC 活動，設定豐富獎勵機制，鼓勵用戶分享。要求在 5月5日前完成活動方案設計與合作網紅/KOL篩選。 設立「社群內容孵化基金」RMB 50 萬元，用於資助高質量粉絲二創內容，並提供官方平台曝光機會，預計在 5月31日前完成第一批內容資助與推廣。

Mofusand 治癒系形象持續深化，跨界聯名成果豐碩

▼

Mofusand 在中國大陸市場憑藉其可愛且具治癒感的鯊魚貓形象，已累積大量忠實粉絲。小紅書上 #mofusand 相關話題瀏覽量月增 18%，特別是與「療癒系」內容結合的帖文互動率增加 25%。 近期在中國大陸的快閃店銷售額顯示，周邊商品，特別是毛絨掛件，單店週銷售額達 50 萬元人民幣，成功證明其市場號召力及與茶飲品牌「奈雪的茶」等聯名活動的影響力。

對蕪解的影響：

Mofusand 的市場表現穩健，尤其在治癒系IP領域擁有核心競爭力。成功的跨界聯名案例，為蕪解創意提供了明確的合作夥伴選擇與營銷策略參考，可有效拓展其周邊商品的銷售渠道與消費群體。

建議行動及截止日：

持續監測 Mofusand 在小紅書與抖音的社群互動數據，重點分析熱門內容類型與用戶參與度。5月1日前提交月度社群數據分析報告。 啟動針對Mofusand與中國大陸前三家知名茶飲品牌、生活風格品牌的聯名合作洽談，目標在 6月30日前簽訂至少一項合作協議。

IP 周邊商品的「體驗化」趨勢明顯

▼

盲盒市場的持續熱潮不僅來自於抽獎的刺激感，更在於產品拆封後內容物的驚喜以及收藏的樂趣，這使得盲盒被視為下一波文創產業的重要動能。 日本市場出現的互動機器人掛飾「Mirumi」預示著 IP 商品從「擁有」轉向「體驗」的趨勢，其「會動」的特性非常適合短影音分享，並在情緒消費市場中佔據優勢。 這暗示了未來商品設計需更加注重互動性與情境感。

對蕪解的影響：

蕪解創意現有盲盒產品已具備初步的「體驗」屬性（抽獎與收藏）。此趨勢提示，未來開發新產品時，需將互動性、多媒體內容連結（如AR濾鏡、社群分享功能）或小型電子組件納入考量，以提升用戶黏著度及社群傳播力。

建議行動及截止日：

設計至少 3 款具互動功能的 IP 周邊商品概念原型，例如結合音效、燈光或簡單動作的掛件。5月30日前提交設計報告與市場潛力評估。 研發團隊立即啟動 AR 濾鏡或小程序應用程式的技術調研，評估將其整合至蕪解 IP 周邊產品的可能性與成本，以提升開箱體驗。6月15日前完成初步可行性報告。

Snoopy 全球 75 週年慶典活動帶來品牌勢能提升

▼

隨著 2025 年「Peanuts」誕生 75 週年，全球各地展開了一系列紀念活動。日本的 Snoopy Museum Tokyo 在 2026 年 3 月啟動「ふかふかミュージアム」新企劃，並推出「MONTHLY PEANUTS GANG」活動，透過互動體驗和限定周邊，持續吸引大量親子及粉絲客群，帶動了相關商品銷量成長 10%。 這表明經典 IP 透過創新策展與互動，依然能煥發新活力，維持穩定的市場熱度。

對蕪解的影響：

經典 IP 如 Snoopy 的成功案例，證明即使是歷史悠久的品牌，也能透過創新內容和體驗式營銷重塑活力。這為蕪解創意旗下如 Maniani World 等有潛力的長線 IP 提供了寶貴的借鑒經驗，指導如何策劃周年慶或主題活動以維持品牌相關性與市場吸引力。

建議行動及截止日：

分析 Snoopy Museum Tokyo 的「ふかふかミュージアム」與「MONTHLY PEANUTS GANG」活動的詳細數據與用戶反饋，製作一份創新策展案例分析報告。5月10日前提交。 為 Maniani World 策劃一個為期一年的「主題季節活動」方案，融合線上互動與線下限定周邊，目標在 7月1日前提出完整的企劃書與預算。

AI 在社群內容生成與消費者互動中的角色演變

▼

儘管消費者偏好「人味」內容，但 AI 正日益成為創作者調整內容以符合本地市場、預測趨勢及提供個人化推薦的強力工具。 愛奇藝等平台已宣布將轉型為非中心化社交媒體，賦予創作者更多 IP 資產與私域流量，這可能改變 IP 內容的創作與分發生態，值得密切觀察 AI 輔助下的「人類創作」新模式。

對蕪解的影響：

AI 在內容生成和趨勢預測方面的潛力，可提升蕪解創意在IP內容創作和社群營銷的效率。愛奇藝的轉型預示著「創作者經濟」與「私域流量」的重要性將進一步提升，蕪解需準備好在這種新生態下，賦能與合作 IP 創作者。

建議行動及截止日：

技術部門立即調研至少 3 款 AI 內容生成工具（例如 ChatGPT-4, Midjourney, Stability AI），評估其在 IP 概念圖、文案創作及社群內容產出中的應用潛力與成本效益。5月20日前提交評估報告。 市場部啟動對愛奇藝非中心化社交媒體模式的深度觀察，分析其對 IP 內容創作者和品牌營銷的影響，並在 6月30日前提交一份分析報告與潛在合作策略。

▼

對蕪解的影響：

小紅書 Redshop 的推出為蕪解創意開啟了多個國際市場的新零售渠道，特別有利於 Maniani World 和 Opanchu 這類在亞洲市場已有基礎的 IP。抓住這一機遇，能顯著擴大品牌國際影響力與銷售額，實現快速市場滲透。

建議行動及截止日：

立即指派專人與小紅書 Redshop 招商團隊建立聯繫，獲取平台詳細政策、費用結構與技術對接資料。5月15日前提交合作潛力評估報告。 針對 Maniani World 與 Opanchu 兩個 IP，制定 Redshop 平台上的產品清單與定價策略，考慮各地市場差異與物流成本。6月10日前完成準備，確保平台一上線即可進行商品上架。
