1. 香港國際授權展揭示「情緒經濟」與數碼化趨勢，盲盒與收藏品持續熱門

於2026年4月27日至29日在香港會議展覽中心舉行的第23屆香港國際授權展 (HKILS) 成功落幕，匯聚了來自香港、中國內地、日本、新加坡、韓國、澳門、泰國及台灣等地的330多家參展商，展示超過600個品牌及授權項目。展會深入探討「情緒經濟」如何重塑授權產業的未來，強調IP透過角色設計、故事內容及情感連結，與消費者建立深層關係的能力。此外，展會還引入了IP及電子商貿支援服務專區，以配合港商在跨境電商平台競爭力提升的需求。值得注意的是，「Taiwan IP Showcase」也集結了近10個高潛力原創角色品牌參展，搶攻國際授權商機。

具體數據/來源：

  • 香港國際授權展 (HKILS) 於2026年4月27-29日舉行，吸引逾330家展商、600多個品牌與IP參與。
  • 會議議題涵蓋全球授權業趨勢及「情緒經濟」對產業的影響。
  • 「Taiwan IP Showcase」集結近10個台灣原創IP參展。

對蕪解的影響：

蕪解創意的IP（如Chiikawa、Mofusand、Opanchu、Maniani World、飯糰與梅子、Muzik Tiger）多以可愛、療癒的形象為主，在建立「情緒價值」方面具有先天優勢。香港作為進入中國大陸市場的門戶，其展會趨勢對蕪解的產品開發與市場策略具有重要參考價值。電子商貿支援服務專區也顯示線上銷售與IP結合的重要性。

2. 蕪解授權 IP Snoopy 在台灣市場動作頻頻，強化在地化連結

國際知名IP Snoopy (Peanuts) 在台灣市場持續拓展，透過大型在地化合作案深化品牌影響力。台中市政府交通局宣布自2026年第一季起與Peanuts合作，將Snoopy及其經典角色融入人行號誌燈、公車候車亭、捷運站、停車場及校園周邊，打造城市級的交通安全視覺化計畫，預計全年執行。此外，高雄義大皇家酒店也正式宣布將於2026年夏季推出全新的Peanuts主題度假娛樂園區，包含35間主題客房、特色餐廳及400坪的家庭娛樂中心。華山1914文創園區亦有Snoopy x 港式奶茶快閃店，活動至2026年4月6日。

具體數據/來源：

  • 台中市政府交通局與Peanuts合作，自2026年第一季起分階段導入Snoopy主題視覺於城市交通設施。
  • 高雄義大皇家酒店將於2026年夏季推出Peanuts主題度假娛樂園區，規劃35間主題客房及400坪家庭娛樂中心。
  • 華山1914文創園區「Snoopy Peanuts Tea Pop-up」活動於2025年12月12日至2026年4月6日舉行。

對蕪解的影響：

台灣市場對Snoopy IP的接受度與市場活絡度極高，多個城市級別的大型合作案將為Snoopy帶來前所未有的曝光與話題性。這對蕪解創意在台灣市場推廣Snoopy周邊商品是絕佳的契機，可藉由IP熱度帶動周邊商品銷量。

建議行動及截止日：

  • 立即啟動台灣市場的Snoopy在地化行銷計畫，包括與台中、高雄相關旅遊/生活媒體合作進行商品宣傳。截止日：2026年05月15日。
  • 與台灣代理商討論，規劃至少2款配合Snoopy主題園區及交通安全宣導的限定款盲盒或掛件商品，並爭取於2026年06月30日前推出。
  • 針對高雄義大主題度假園區的開幕，協調台灣合作夥伴，規劃一系列結合園區主題的行銷活動或獨家商品，以搶佔親子旅遊市場。截止日：2026年07月31日。

3. 中國大陸盲盒市場規模持續增長，但國產IP出海仍有巨大潛力

中國大陸盲盒市場持續升溫，據市場機構統計，2025年中國盲盒市場規模已達人民幣580億元，近年來一直保持20%以上的年均增速，成為全球最大且增長最快的市場。然而，有2026年趨勢報告指出，全球最賺錢IP的商品收入占比超過60%，但中國IP僅佔2%，顯示中國本土IP在全球市場的拓展仍有巨大空間。

對蕪解的影響：

蕪解有多個授權IP在中國大陸市場（Chiikawa、Mofusand、飯糰與梅子、Muzik Tiger），應持續深耕並把握市場增長機會。同時，應思考如何協助這些IP商品出海，將中國大陸市場的成功經驗複製到國際市場，搶佔更多全球IP商品市場份額。

建議行動及截止日：

  • 分析中國大陸市場上熱銷盲盒IP的成功出海案例，研究其海外市場定位及推廣策略。截止日：2026年06月15日。
  • 於2026年06月30日前提交一份針對Chiikawa和Mofusand出海潛力的初步評估報告，包含目標市場分析與建議推廣渠道。

4. 香港玩具展首設「潮玩館」，凸顯潮玩與盲盒市場熱度

對蕪解的影響：

「潮玩館」的設立直接反映了市場對設計師玩具和盲盒的強勁需求，且公眾開放策略有助於擴大消費群體。蕪解作為IP周邊商品批發/零售商，應密切關注此類展會，以發掘新的設計師IP、合作機會及最新的潮流趨勢，尤其是在亞洲市場。
