038 週執行時間表

每週 5-10 小時投入，不是全職。每週都有明確的「完成標準」—— 達到了才往下一週走，沒達到就停在這週繼續調。

Week 1 註冊工具 + 跑通第一張圖 投入 5 小時
  • 註冊 Replicate（送 $1 免費額度）+ 綁信用卡
  • 訂閱 Claude Pro（US$20/月）或 Max + 安裝 Claude Code
  • 挑 1 個熱賣 SKU 的淘寶原圖當測試
  • 用 Replicate 網頁版手動跑：去水印 → 換模特 → 換背景，看品質
完成標準：得到 1 張你自己滿意的 AI 生成商品圖（不用上架，純測品質）
Week 2 prompt 調教 + 確定 5 個模特風格 投入 8 小時
  • 用 Claude Code 跟 AI 對話：「我要的模特風格是 XX」→ AI 幫你寫 prompt template
  • 定 5 個常用模特風格（如：日系少女 / 韓系氣質 / 歐美街拍 / 港風 / 文青）
  • 每個風格存一份 prompt + 範例圖
  • 跑 5 個 SKU 各 5 風格 = 25 張，挑出最好的 5 張
完成標準：5 個風格的 prompt template 寫好，每個都有範例證明 prompt 穩定可重複
Week 3-4 批次腳本化 — 一鍵跑 20 張 投入 15 小時
  • 跟 Claude Code 說：「幫我寫一個 Python script，輸入資料夾 → 跑完整 pipeline → 輸出」
  • 整合 Replicate API（不再用網頁版手動）
  • 加上輸出規格：每張圖出 5 個風格版本
  • 跑一次 20 個 SKU 的 batch 測試效能 + 成本
完成標準：命令列輸入「python batch.py ./sku_folder」就能跑完 20 個 SKU 出 100 張圖
Week 5-6 接 SHOPLINE API — 一鍵上架 投入 15 小時
  • 註冊 SHOPLINE Developer 帳號 → 建 Private App → 拿 API key
  • 用 Claude Code 寫上傳腳本：跑完 pipeline → 自動 PUT 商品圖到 SHOPLINE
  • 加上「主圖 / 細節圖 / 模特圖」分類標籤
  • 建簡單 web UI（或用 Notion / Airtable）讓你挑要用哪個版本
完成標準：從淘寶圖到 SHOPLINE 上架，全程不用打開 Photoshop
Week 7-8 全自動觸發 + 異常監控 投入 12 小時
  • SHOPLINE 新品 webhook → Cloudflare Worker 接收 → 觸發 pipeline
  • 跑完後 LINE / 飛書 通知你「5 版本好了，去挑」
  • 加異常監控：跑失敗自動 retry / 通知
  • 建簡易 dashboard 看本月跑了幾張、花了多少 Replicate 額度
完成標準：你上架新品時，不用做任何圖片相關工作 —— 系統自動準備好 5 版本給你挑

這 4 件事加起來 90 分鐘內可以做完，做完你就在 Week 1 的軌道上了。
  1. 註冊 Replicate（5 分鐘）— 用 GitHub 帳號登入，綁信用卡，會送你 $1 免費額度（夠跑 100+ 張測試圖）
  2. 訂閱 Claude Pro + 裝 Claude Code（15 分鐘）— claude.com 訂閱 → 跟著官方文件裝 CLI，第一次跑 claude 進入對話
  3. 挑 5 個熱賣 SKU 的淘寶原圖（20 分鐘）— 存成資料夾，每個 SKU 一個子資料夾。這是你 Week 1-2 的測試素材
  4. 跟 Claude Code 講這份計畫（30 分鐘）— 把這份網頁的 URL 丟給 Claude Code 說「我要做這個，從 Week 1 開始」，它會帶你跑

08誠實 flag — 你會踩到的雷

1. AI 換模特品質會波動。同一個 prompt 跑 10 次，可能 2-3 次手指畸形、表情詭異、商品 logo 變形。這就是為什麼要出 5 個版本給你挑，不要妄想一張就用。
2. 淘寶反爬風險。抓淘寶原圖太頻繁會被封 IP。建議：用淘寶買家中心的訂單詳情圖（你已經是買家有權限）；或限速每分鐘 5 張；或用第三方代抓服務。
3. SHOPLINE API rate limit。免費版每分鐘 30 次請求。批量上架 100 個 SKU 要 30+ 分鐘，不能心急一次推完。
4. 法律邊界 — 不要用真人模特生成圖侵權。AI 換模特生出來的「臉」如果太像某個真人明星 / 模特，可能侵權。用 Replicate 內建的虛擬模特模型較安全，不要用「指定臉」的 face swap 模型。
5. 一個常被忽略的點 — 商品細節圖不要用 AI 生。主圖、模特圖可以 AI 生，但「商品材質特寫」「五金細節」「縫線」這些必須是真實照片，不然客戶收到貨會說「跟圖不一樣」退貨爆量。細節圖建議用淘寶原圖去水印就好，不要過度 AI 處理。
