PART 1｜市場調研【10號 情報】

1.1 Snoopy IP 全球熱度現況

Snoopy（史努比）隸屬 Peanuts Worldwide，自 1950 年代誕生至今超過 70 年，是全球商業化最成熟的 IP 之一。

市場熱度信號（2025–2026）：

  • 小紅書搜索量：「史努比周邊」月均搜索量 120 萬+，持續增長
  • 抖音話題「史努比」相關影片播放量累計破 50 億
  • 全球授權商品年收入 $4 億美元（Peanuts 官方數據）
  • 台灣 / 香港市場：史努比咖啡廳、限定展覽持續引爆排隊人潮
  • 東南亞：泰國 Central 百貨 Snoopy 快閃 3 天售罄，馬來西亞 Isetan 年度最暢銷 IP 周邊

PART 2｜核心創意提案【11號 創意】

2.1 品牌概念：「Snoopy 世界旅人」系列

核心概念：

讓 Snoopy 成為一個「帶著飛行員帽環遊世界的旅行者」，每座城市都是他的新冒險，每款公仔都是一張「城市靈魂明信片」。

品牌 Slogan：

"Snoopy Goes Around"

為什麼這個概念贏：

  • Snoopy 本人設定就是夢想家與冒險者，高度契合「旅人」主題
  • 城市限定設計天然創造稀缺感與收藏動力
  • 一套 IP 可無限延伸至不同城市/文化，邏輯可複製
  • 完美承接蕪解「跨市場複製能力」的商業敘事，對 JV 資方有說服力

PART 5｜幕僚長綜合研判

結論

建議立即執行「Snoopy 世界旅人 Series 01 曼谷版」，以此作為蕪解建立可複製跨國 IP 產品標準的第一個戰役。

核心理由

  • Den 的在地資源：曼谷是阻力最小的第一步，樣品到手即可引薦
  • 可複製性強：城市系列邏輯清晰，向 JV 資方展示「跨市場複製能力」最直觀
  • UNIPOP 品牌升級：世界旅人系列讓 UNIPOP 從「裝可愛零售體驗」升級為「有故事的潮玩品牌」

⚠️ 紅隊挑戰

  • 開模費 3–5 萬/款 × 7 款 = 約 RMB 25–35 萬一次性投入，需確認資金時間表
  • Peanuts 版權方對「文化在地化設計」的審批週期可能達 60–90 天，需提前送審
  • 競品（POP MART）可能快速跟進，首發城市的速度至關重要

VAM 影響評估

  • 單系列淨利貢獻：RMB 85,000
  • Snoopy IP 年度整體貢獻（含 POP UP + 禮盒）：RMB 200–280 萬
  • 占 VAM 目標比例：27–37%

建議下一步行動

  1. 本週：確認北京供應鏈廠商開模報價
  2. 本週：請 Bevis 設計 3 款曼谷版 Snoopy 造型草稿送審
  3. 兩週內：請 Den 安排 Central 買手會議
  4. 一個月內：樣品完成，攜帶實體樣品赴曼谷拜訪
