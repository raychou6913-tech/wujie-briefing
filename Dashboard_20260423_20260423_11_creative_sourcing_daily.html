引領潮玩市場新趨勢

🌟 本日創意焦點：零售陳列 + 體驗行銷

在競爭激烈的 IP 周邊市場，單純的商品陳列已無法有效吸引消費者。今日報告將聚焦於如何透過創新的零售陳列與互動體驗，讓蕪解創意旗下的正版授權 IP 商品在眾多選擇中脫穎而出，不僅刺激消費者的購買慾望，更深化品牌與 IP 形象的連結，進而實現我們的 VAM 目標。

🎯 今日主推提案

IP主題情境盲盒專區

將貨架轉化為沉浸式 IP 世界，提升消費者停留與購買意願。

預估市場吸引力：
★ ★ ★ ★ ★

靈感來源

觀察到 Pop Mart 等盲盒品牌在店鋪中常設置大型 IP 主題裝置藝術與陳列，營造出強烈的品牌氛圍。例如，Labubu 的某些系列會搭配特定情境裝置，讓收藏品更具生命力。迪士尼商店更是將 IP 情境化陳列的極致體現，讓顧客置身於魔法世界。

具體商品規格

  • 核心商品： 蕪解創意旗下 5cm/8cm 盲盒公仔（例如：Snoopy 奇幻冒險系列、Chiikawa 森林日常系列）。
  • 陳列裝置：： 模組化設計，可根據零售空間大小彈性組合。材質以環保木材、壓克力搭配高解析度 IP 主題背板及地貼。可加入小型互動裝置（如按鈕發光、簡單音樂播放）。
  • 示範 IP：： 首波主推 Snoopy/Peanuts（全球市場）與 Chiikawa（中國大陸市場）。

目標市場與客群畫像

  • 目標市場：： 中國大陸、台灣、香港、泰國、馬來西亞、新加坡、歐洲、紐澳等 IP 粉絲眾多的零售通路。
  • 客群畫像：： 18-35 歲的年輕潮玩愛好者、IP 忠實粉絲、尋求新鮮購物體驗的消費者、家庭親子客群。

預估零售定價 + 批發毛利率

  • 盲盒公仔：
    • 零售定價：RMB 79/件 (約 USD 10.82)
    • 預估批發價：RMB 47.45/件 (約 USD 6.5) (按 40% 零售商毛利率計算)
    • 預估生產成本：RMB 7/件 (以 8cm 公仔計)
    • 批發毛利率 = 85.2%
  • 陳列裝置：： 作為行銷工具，初期以成本分攤或租賃模式與零售商合作。

VAM 貢獻潛力（具體數字估算）

假設於 Y1 在各主要銷售市場選定 50 個高潛力零售點推廣，每個點位平均每月可帶來 200 個盲盒的增量銷售。年度增量銷售總量：120,000 件。預估可為年度 VAM 目標貢獻約 人民幣 485.4 萬元增量毛利。

上市時程建議

2026 年 Q3 於中國大陸、香港、台灣市場啟動首批 10 個示範點的陳列區建置與測試，並同步評估東南亞市場的潛力通路。

目標市場：亞洲, 歐洲, 紐澳 商品類型：盲盒公仔, 店內裝置 VAM潛力：高

盲盒互動尋寶體驗

讓購物成為一場 IP 冒險遊戲，透過遊戲化提高消費者參與度。

預估市場吸引力：
★ ★ ★ ★ ★

靈感來源

借鑒近年流行的「劇本殺」、「密室逃脫」等沉浸式遊戲體驗，以及 Pokémon GO 等 AR 互動遊戲概念。Chiikawa 在日本常舉辦結合插畫與解謎的快閃活動，吸引粉絲參與。透過遊戲化提高消費者參與度與停留時間。

具體商品規格

  • 核心商品：： 蕪解創意所有 IP 授權的 5cm/8cm 盲盒公仔。
  • 配套商品：：
    • 「IP 尋寶任務日誌」： 一本輕薄的任務手冊（約 A5 尺寸，20-30 頁），內含 IP 背景故事、尋寶線索、貼紙、印章收集頁及商品折扣券。材質為再生紙，封面可選不同 IP 設計。
    • 店內尋寶元素：： 設有 IP 主題 QR Code 提示牌、隱藏式 IP 標誌或小公仔模型於店內各處。

目標市場與客群畫像

  • 目標市場：： 中國大陸、香港、台灣、泰國等具備大型購物中心或百貨公司的高人流區域。
  • 客群畫像：： 喜歡互動遊戲的年輕族群、親子家庭、渴望新奇體驗的 IP 粉絲。

預估零售定價 + 批發毛利率

  • 盲盒公仔：： (同提案一) 零售定價 RMB 79/件，批發毛利率 85.2%。
  • IP 尋寶任務日誌：：
    • 零售定價：RMB 19/本 (約 USD 2.6)
    • 預估批發價：RMB 11.4/本 (約 USD 1.56) (按 40% 零售商毛利率計算)
    • 預估生產成本：RMB 2.5/本
    • 批發毛利率 = 78.1%

VAM 貢獻潛力（具體數字估算）

假設 Y1 舉辦 20 場主題尋寶體驗活動，每場平均吸引 1,000 名參與者購買任務日誌，並平均每位參與者額外購買 1.5 個盲盒。預估可為年度 VAM 目標貢獻約 人民幣 139.15 萬元增量毛利。

上市時程建議

2026 年 Q3-Q4 於中國大陸、香港、台灣的主要購物中心或百貨公司合作，推出首波尋寶體驗活動，並依據反饋逐步優化與擴展至東南亞市場。

目標市場：亞洲(高人流區) 商品類型：盲盒公仔, 任務日誌, 體驗活動 VAM潛力：中高

💡 延伸創意點

  • 盲盒開箱體驗區

    在零售店內設置專屬的開箱區域，配備 IP 主題背景牆、補光燈及手機支架，鼓勵消費者現場開箱並拍攝分享至社群媒體。

    具體可執行的下一步： 由創意選品部與市場部合作，設計 3 款不同 IP（Snoopy, Mofusand, Muzik Tiger）的開箱體驗區標準模組，並於 2026/05/20 前完成成本估算。

  • AR 互動商品導覽

    在關鍵商品陳列架旁設置 QR Code，消費者掃描後可透過手機 AR 濾鏡，看到盲盒公仔在現實環境中的 3D 虛擬形象，或與 IP 角色進行趣味合影。

    具體可執行的下一步： 由數位創新組負責調研至少 3 家 AR 技術開發商，並於 2026/05/30 前提交一份初步的可行性分析報告。

  • 期間限定 IP 快閃店中店

    與大型連鎖店（如便利店、超市）合作，推出小型的 IP 快閃「店中店」模式，以限定商品或主題活動吸引人流。例如，針對 Opanchu 設計「打工人加油站」主題快閃。

    具體可執行的下一步： 由市場部與銷售部於 2026/06/15 前，針對 Opanchu 與 Maniani World 兩個 IP，完成兩份「快閃店中店」概念企劃書及潛在合作夥伴清單。

🏪 競品參考

泡泡瑪特 (Pop Mart)

作為盲盒市場的領軍者，泡泡瑪特非常擅長通過其門店設計、大型 IP 裝置藝術以及主題性櫥窗陳列來吸引消費者。他們經常與知名藝術家和 IP 合作，將盲盒系列融入沉浸式的故事場景中，例如「Mega Space Molly」的宇宙主題空間。

吉伊卡哇 (Chiikawa) 官方快閃店

在日本及中國大陸市場，Chiikawa 的官方快閃店往往營造出高度還原漫畫情境的空間，提供豐富的拍照打卡點和限定商品，成功將線上人氣導流至線下實體店，創造粉絲的排隊熱潮。

Sanrio/Snoopy 專賣店

這些經典 IP 的專賣店在全球範圍內都將品牌視覺與商品陳列緊密結合，通過色彩、燈光和道具，打造出充滿 IP 氛圍的購物環境，不僅販售商品，更販售一種 IP 生活風格。

迪士尼商店 (Disney Store)

迪士尼是體驗式零售的典範。其門店不僅是商品銷售點，更是 IP 故事的延伸，通過互動裝置、主題佈景和角色扮演活動，為顧客提供獨一無二的沉浸式體驗，極大地提升了商品的吸引力。
