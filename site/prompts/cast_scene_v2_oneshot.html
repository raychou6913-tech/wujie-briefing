═══ 任務 ═══

生成 1 張完整的 UNIPOP CAST 世界觀大合照，8 隻角色全部出場 + POPS 散落。Aspect ratio **21:9 寬版面**（cinematic letterbox）。

═══ 場景 · 1980s Tokyo 後巷深夜 ═══

濕潤柏油巷子，霓虹便利商店招牌（漢字 + 假名）在背景發光。中央偏右一台老式日本自動販賣機（紅白復古 + 飲料罐隱約可見），燈光從內透出暖黃。地面反射粉紫霓虹光。電線桿 + 電線交錯。夜空暮光紫漸層（#2A1F45 → #1A1428）。氛圍：Hiroshi Nagai 永井博 City Pop × Studio Ghibli 夜色。

═══ 8 隻 cast 安排（左到右 / 前到後）═══

1. **GARLIC**（最左，前景）— 在路燈下單手轉籃球，自以為帥的笑容，紅色 jersey + peach 短褲，BBcall 別腰間。大蒜頭（白色 + 紫色 papery 紋）。

2. **TOFU**（左半中段）— 靠在自動販賣機旁的牆上，手拿 Pocari Tears 飲料罐喝，墨鏡反射霓虹（淚滴從墨鏡下沿）。豆腐方頭 + slim suit，金魚在豆腐裡若隱若現。

3. **NOKO**（中前景偏左，蹲坐在低矮木箱或階梯上）— 藍色補丁 plush 身、一耳縫線、一眼鈕扣，懷裡抱**另一隻**更小更破舊的灰色 abandoned 玩偶（不是 POPS！）。氛圍：被遺棄物的共生者，安靜看著大家。

4. **MIKA**（中前景偏中）— 站著拿 Polaroid 相機對巷子拍照，紫 bob + 葡萄頭飾（其中一顆是眼球！必須畫出來），pink 上衣 + 紫短褲。

5. **BEEPO**（中段，便利商店門口階梯上）— 坐著玩 Game Boy，雙腿屈起。Mac 1984 螢幕頭 + Galaga 蜜蜂瞳孔 + floppy disk 嘴。

6. **NOVA**（中後段，BEEPO 旁邊往右後）— 仰頭看夜空，月亮頭剛好跟天空的月亮形成「她頭就是月亮」的視覺呼應。手拿塔羅卡 + 水晶 pendulum，星空圖案 dress。

7. **SOLO**（中後段中央 / 安靜站著）— 紫 bob + sun-gold 貝雷帽 + 白口罩 + navy 雙排扣 blazer。**右肩站著 sun-gold POPS**，POPS 嘴張開喊話，旁邊白色 speech bubble 寫 "?!" 或 "Hello!"。SOLO 眼神略尷尬。

8. **MOMO**（最右）— 站在 MIKA 後方一點，雙 A 電池頭，拿著紅色電線插進販賣機旁牆上插座給自己充電。另一手拿小牛奶盒。
